# 변경 내역 (Changelog)

이 파일은 배포마다 갱신한다. 태그 이름 규칙: 안정 `vX.Y.Z`, 개발/테스터 `vX.Y.Z-dev.N`.
최신이 위로 온다.

## v1.0.6-dev.11 (2026-07-06) — 프리릴리스(테스터용)
- **어린 사과나무**: 독버섯처럼 보이던 묘목을 나무 새싹(줄기+뾰족한 잎)으로.
- **개발자 이름 이스터에그 테마**: 김민욱=고라니→돼지, 서태양=달→웃는 태양, 박서현=돌→모아이.

## v1.0.6-dev.10 (2026-07-06) — 프리릴리스(테스터용)
- **당근 수확 난이도↑**: 한 번에 안 뽑히고 여러 번(약 3회) 끌어당겨야 뽑힘.
- **이해도 표시 정리**: 어긋나 보이던 달 아이콘 제거, 단계 이름 안 잘리게.
- **달 이스터에그**: 활성화 시 제목이 '악)몽중농원'으로, 작물 선택창 악몽 체크도 기본 ON.
- **넘기기 게이지 축소**: 크레딧에서 스토리 글자를 덜 가리게.
- **버전 표기**: 박스에 넣어 잘 보이게(디버그용 dev 버전 표시).

## v1.0.6-dev.9 (2026-07-06) — 프리릴리스(테스터용)
- **완전 초기화**: 이제 진짜 태초로(악몽 모드·회차 기록까지 리셋), 실수 방지 위해 '초기화' 입력 요구.
- **잠긴 업적도 이름 표시**: 조건은 비밀이되 무엇을 노릴지 유추 가능.
- **버그**: 업적 탭에서 이야기 항목이 눌리던 문제, '사과나무 반찬' 어색한 문구 수정.
- **개발자 이름 이스터에그**(서태양·김민욱·박서현), '아버지의 하루' 출현 빈도 낮춤.
- 갤러리 별 잇기 감상 후 이해도 획득 장면 생략.

## v1.0.6-dev.8 (2026-07-06) — 프리릴리스(테스터용)
- **별 잇기 다시 하기**: 갤러리 이야기·기억 탭에서 별 잇기 미니게임을 다시 플레이(끝나면 갤러리로 복귀).

## v1.0.6-dev.7 (2026-07-06) — 프리릴리스(테스터용)
- **갤러리에서 이벤트 다시 하기**: 이야기·기억 모달에 '다시 하기' 버튼 — 그 이벤트 미니게임을 다시 플레이.
- **무논에 흙 노출**: 벼 논에서 모마다 진흙 둔덕이 물 위로 봉긋 드러나게.
- **벼 물고임 경고 정리**: 관개농업(고인 물)은 정상이라 "물이 고여 있다" 경고를 벼에는 띄우지 않음.
- **호미 그래픽**: 녹 닦기 미니게임을 실제 호미(자루+쇠날)에 녹슨 자국으로.
- **돌 다양화**: '쓰러진 허수아비'의 돌을 모양·크기·색이 제각각인 여러 종류로.

## v1.0.6-dev.6 (2026-07-06) — 프리릴리스(테스터용)
- **당근 수확 방식 변경**: 연타 → 잡고 위로 쭉 끌어올려 뽑기(잡초 뽑듯). 놓치면 다시 잡을 수 있음.
- **해충 디자인**: 크고 징그럽던 굼벵이를 예전의 작은 딱정벌레로. 벌레 잡기 미니게임도 같은 모습으로 통일.
- **이스터에그**: 타이틀에서 달을 4번 두드리면 악)몽중농원 모드가 켜지고 배경·음악이 검붉게 바뀜.
- **악)몽중농원**: 인트로·타이틀 배경도 검붉게.
- 엔딩에서 '다른 엔딩 다시 보기(1~7)' 삭제. 확인창에서 **Enter로 '예'** 선택 가능.
- 업적 설명 텍스트가 칸 밖으로 나가던 것, 이해도 단계 이름이 패널을 넘던 것 수정.

## v1.0.6-dev.5 (2026-07-06) — 프리릴리스(테스터용)

## v1.0.6-dev.3 (2026-07-06) — 프리릴리스(테스터용)
### 버그 수정
- **벼 수확 데드락**: 탈곡 중 마우스를 놓으면 다시 잡지 못해 나머지 수확이 막히던 문제 수정
  (손을 떼면 '준비' 상태로 복귀). 도정 단계 잠재 버그도 함께 방어.
### 추가/개선
- **ESC로 설정 열기**: 게임 중 ESC로 설정 오버레이가 열림(작물선택·이름입력의 뒤로가기 및
  확인/종료창 ESC는 그대로 유지).
- **물주기/물대기 연출**: 물뿌리개 입구(스파웃)가 붓는 동안 아래로 기울며 물줄기가 내려간
  입구에서 쏟아지고, 논 물꼬는 판이 열린 문턱에서 물이 쏟아지도록 애니메이션 추가.
- **벼 UI 리워크**: 물 위에 깔리던 갈색 흙 패치 제거, 막대 같던 벼를 밑이 굵고 끝이 휜 잎사귀와
  고개 숙인 황금 이삭으로, 논 물에 깊이감(수면막·엇갈린 물결·반짝임) 부여, 밥공기 낟알 대비↑,
  성장바 벼 아이콘을 초록 벼 포기로. 악몽 모드 색도 반영.

## v1.0.6-dev.2 (2026-07-03) — 프리릴리스(테스터용)
- **돌발상황 전용 BGM** 신규(numpy 합성) — 밭 정리·선택형 이벤트에 적용.
- **벼 논**: 밭 전체가 물에 잠기는 무논으로, 수분량 실시간 반영. **모내기**(처음부터 어린 모).
- **감자 포기**: 둥근 잎 덤불 + 감자꽃. 성장바 아이콘 감자화(체리 탈피).
- **사과나무**: 거름 주기(흙 북돋기) 행동이 항상 보이도록 정리.
- 픽셀 나무 '죽순 단계' 보정(끝눈), 밥공기 그래픽, 굼벵이 애벌레 디자인.
- 악)몽중농원: 논·나무밭·별잇기 배경 검붉게. 이해도 라벨 명확화("마음"→"이해도").
- 사과 '반찬' 표현 수정. 엔딩 크레딧: 화면전환 없이 이어서 + 스페이스바 홀드 게이지.

## v1.0.6-dev.1 (2026-07-03) — 프리릴리스(테스터용)
- **업적 시스템** 신설(마인크래프트 도전과제 풍): 해제 토스트 + 갤러리 "업적" 탭(9종).
- **픽셀 나무 리워크**: 가지가 먼저 나오고 잎이 나중에 돋는 현실 성장, 열매, 악몽 검붉은 잎.
- **나무 가지치기 QTE**: 가지가 실제 나무에서 뻗어 나오게.
- **새벽의 고라니 QTE**: 진짜 울타리+부러진 틈+고라니. 길 잃은 벌: 목적지 꽃 모양.
- **감자 수확**: 좌우로 번갈아 흔들어 캐기(풀림 게이지).
- 작물 플레이타임 차등(사과 34/벼 26), 작업표시줄 아이콘 대응.
- 이름 입력 2회차 버그, 갤러리→게임 진입 버그, 각종 겹침/여백 수정.
- 작물 선택창 클리어 횟수 배지, 저장/불러오기 확인 절차.
