## v2.2.2-dev.12 (2026-07-14) — 프리릴리스 (영어 현지화 QA: 긴 문장 넘침·미번역 수정)
- **밭일 일지 완역 + 넘침 해결**: 일지를 **한국어 원문으로 저장하고 표시 시점에 번역**하도록 정리해,
  엔딩에서 언어를 바꿔도 즉시 반영되고 창을 넘치지 않습니다. `[N일째 · 계절 · 날씨]`·`성장 N%` 같은
  동적 줄은 패턴을 인식해 현재 언어로 재조립하고, `", "`로 이어 붙인 '조합' 줄(잡초·벌레·배수)은 조각별로
  번역해 다시 잇습니다. 긴 영어 줄은 패널 폭(560px)에 맞춰 자동 줄바꿈합니다
  (`ending._localize_journal_line`, `farm_simulator._write_journal`).
- **엔딩 '당신의 태도'가 영어로**: 노멀/트루 엔딩 결과 화면의 태도 요약이 한국어로 남던 문제를 고쳐,
  접두어와 항목(인내·세심함·공감·회복력)을 각각 번역해 표시합니다(`ending._draw_result`).
- **인트로/악)몽중농원 서사 세로 넘침 해결**: 영어가 한국어보다 줄 수가 많아 도입부 텍스트가 패널 위아래로
  삐져나오던 문제를, **페이지마다 패널 안에 들어오는 가장 큰 폰트를 자동 선택**하도록 고쳤습니다
  (`intro.prepare_page`). 한국어·짧은 페이지는 기존 크기 그대로입니다.
- **이름 입력 안내문 자동 축소**: `Your name in the dream field?` 등 긴 영어 안내가 카드를 넘치지 않게
  카드 폭에 맞춰 폰트를 줄입니다(`name_input._fit_font`).
- **설정 토글 라벨 영어화**: `자동저장/전체화면/버전 표시 ON/OFF` 라벨이 영어 카탈로그에서 누락돼 한국어로
  뜨던 것을 채워 넣었습니다(`Autosave/Fullscreen/Version: ON/OFF`).
- **개발자 모드 버튼 정리**: 영어 라벨이 버튼을 넘치던 문제를, 버튼 폭에 맞춰 폰트를 자동 축소하고
  가장 긴 라벨은 짧은 표현으로 바꿨습니다(`악몽 토글`→`Nightmare`, `이해도 +N`→`Insight +N`).
- **종료 확인문 단축**: `Are you sure you want to quit?` → `Quit the game?` 로 줄여 오버레이에 알맞게 맞췄습니다.
- **Crowdin 소스 재생성**: 위 문자열 변경을 반영해 `core/locales/en.json`·`ko.json`을 다시 내보냈습니다.
- 알려진 한계: 당근이 아닌 작물(감자·사과·벼)에서 일지의 작물 명사가 영어로는 아직 'carrot'/원문으로 표시될 수
  있습니다(카탈로그가 리터럴이라 향후 `{crop}` 자리표시자화 필요 — `HANDOFF.md` 참고).

## v2.2.2-dev.11 (2026-07-14) — 프리릴리스 (버그·모바일 수정 배치)
- **언어 전환이 농장 일지에 즉시 반영**: '살펴보기' 결과처럼 조사·상태가 합쳐진 동적 문구를 저장된 채로
  쓰다 보니 언어를 바꿔도 한 턴 늦게 반영됐습니다. 이제 매 프레임 현재 언어로 다시 생성해 즉시 바뀝니다
  (`farm_renderer`).
- **미니게임 점수 가림 수정**: 배속(▶▶)·설정(⚙) 버튼이 미니게임 상단바의 점수 박스 오른쪽을 덮던 문제를
  고쳤습니다. 미니게임 씬에서는 두 버튼을 숨깁니다(`game_main`, `settings_overlay`).
- **흙 북돋기가 작물을 되살리도록 수정**: 잡초·해충 압박이 큰(=작물이 죽어가는) 상황에서 흙 북돋기 회복(+8)이
  같은 턴의 압박(-11)에 상쇄돼 오히려 체력이 깎이던 '죽음의 소용돌이'를 고쳤습니다. 체력이 낮을수록 회복을
  크게 줘(최대 +15) 잘 해내면 확실히 살아납니다(`farm_simulator`). 중복 `SoilMound` 스텁 클래스도 제거.
- **모바일 이름 입력 개편**: 입력칸을 화면 맨 위에 붙이지 않고 **상단 절반에 상하대칭**으로 배치했습니다.
  또 **키보드가 뜨기 전에는 PC처럼 가운데 카드**로, 입력칸을 탭해 키보드를 띄우면 상단 배치로 전환합니다(`name_input`).
- **모바일 앱 아이콘**: 투명 여백을 잘라 아이콘 칸을 **꽉 채운 투명 배경** 아이콘(`core/icon.png`)을 만들어
  `buildozer.spec`에 연결했습니다.

## v2.2.2-dev.7 (2026-07-13) — 프리릴리스 (영어 현지화 기반 + 한/영 전환, 스팀 글로벌 대비)
- **한국어 ↔ 영어 언어 전환 추가**: 설정창에 **언어 토글**(한국어/English)을 넣고 선택을 저장합니다(`user_settings.json`). 스팀 글로벌 출시 대비 첫 단계.
- **현지화 엔진(`core/i18n.py`)**: 원문(한국어)→영어 카탈로그(`core/i18n_data.py`, **약 900개 번역**)를 **렌더 계층에서 자동 적용**합니다.
  - `get_font()` 폰트 래퍼가 `render()`/`size()` 시 문자열을 번역하고, `wrap_text()`는 줄바꿈 전에 원문을 통째로 번역합니다. → 대부분의 라벨·대사가 호출부 수정 없이 번역됩니다.
  - **로직 키(요일·작물·기억 제목 등)는 한국어 원문 그대로** 두고 '표시'만 번역하므로 세이브 호환성·게임 로직에 영향이 없습니다. 번역이 없으면 한국어로 폴백.
  - 변수와 합쳐지는 동적 문자열은 `i18n.tf("… {n} …", n=…)`로 처리 — 타이틀·설정·HUD(요일/날씨/건강/예보)·업적 개수·미니게임 카운터·엔딩 통계·크레딧 등 주요 화면 반영 완료.
- **서사(내러티브) 현지화 완료 — `i18n.tnar`**: 이름 조사(은/는·이/가·을/를)와 작물 치환(당근↔감자↔사과↔벼, 밭↔논↔과수원, 젓가락↔포크↔숟갈)이 얽힌 대사를 영어로 옮겼습니다. **인트로(일반+악몽)·엔딩 5종·회상 9종·수확 안내·미니게임 결과·전환 화면**을 네 작물 모두에서 검증했습니다(예: 벼 플레이 시 "raising the rice … in this hellish paddy").
- **줄바꿈 버그 수정**: `wrap_text`가 영어 줄바꿈 경계에서 **단어 사이 공백을 삼키던 문제**("carrotinto")를 고쳤고, `transition.py`의 자체 줄바꿈도 `wrap_text`로 바꿔 단어가 잘리지 않게 했습니다. 한국어 줄바꿈은 그대로.
- **영어 폭 대응 레이아웃**: 이해도 뱃지가 넘칠 때 단계명을 생략, 타이틀 '오늘 할 일'→'To-Do'로 예보와 겹침 방지.
- **남은 이벤트 문자열까지 마무리**: 돌발 상황(강제 대기)·자세히 보기·아버지의 날·별 잇기 안내·개발자 이스터에그·개발자 모드(F9)까지 번역해, 사실상 전 화면이 영어로 표시됩니다.
- **Crowdin 연동 지원**: 번역을 `core/locales/<lang>.json`(Crowdin 동기화 파일)에서 실행 시 읽도록 했습니다. `core/i18n_data.py`(파이썬 베이스라인) 위에 JSON을 덮어써 갱신하므로, **번역가가 Crowdin에서 고치면 코드 재생성 없이 반영**되고 향후 언어(일/중 등)도 JSON만 추가하면 됩니다. `crowdin.yml`, 내보내기 도구 `tools/i18n_export.py`(en.json은 기존 번역 보존 병합), 빌드 번들(EXE·APK) 포함까지 세팅했습니다.
- 번역 구조·확장·Crowdin 워크플로는 `LOCALIZATION.md`, UI 통일 계획은 `DESIGN_AUDIT.md` 참고.
- **UI 일관성 P1 (픽셀 통일 시작)**: 그동안 당근만 픽셀 스프라이트이고 **사과·감자·쌀밥은 벡터 도형**이라
  이질감이 컸는데, 셋 다 **픽셀 스프라이트로 다시 그려 통일**했습니다(`core/assets.py`). 화면 스케일링도
  1440p·4K처럼 2배 이상 들어가는 해상도에서는 **정수 배율로 스냅**해 도트를 또렷하게 하고, 1080p·720p는
  기존대로 꽉 채웁니다(작아지는 리그레션 없음). 전체 디자인 통일 로드맵은 `DESIGN_AUDIT.md`.
