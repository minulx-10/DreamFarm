# 몽중농원 🥕 (DreamFarm)

> *밀어낸 당근 한 조각에는, 누군가의 새벽이 들어 있었다.*

식탁 위 당근 반찬을 말없이 밀어낸 아이가, 그날 밤 꿈속의 당근밭에서 눈을 뜬다.
**"네가 밀어낸 것을, 이번에는 네 손으로 길러 보아라."**
한 뿌리의 당근을 길러내는 과정을 통해 아버지의 노동과 사랑을 이해해 가는,
Pygame으로 만든 감성 내러티브 농장 게임입니다.

---

## 다운로드 (Windows)

[**Releases**](../../releases) 페이지에서 최신 `몽중농원.exe`를 내려받아 바로 실행할 수 있습니다.
(별도 설치 없이 실행되는 단일 파일. 저장 파일은 exe와 같은 폴더에 생성됩니다.)

## 직접 실행 (개발)

```bash
pip install -r requirements.txt
python main.py
```

- Python 3.10+ / pygame(-ce) / numpy 필요
- 픽셀 폰트(Galmuri11)는 저장소에 포함되어 있으며, 없을 경우 자동 다운로드합니다.
- 오디오 장치가 없는 환경에서도 자동으로 무음 처리되어 정상 실행됩니다.

## 조작법

| 입력 | 동작 |
|------|------|
| 마우스 클릭 | 행동 선택, 드래그(정리·물주기·수확), 대화 진행 |
| 스페이스바 | 텍스트 진행 / 넘기기 |
| **M** | 배경음·효과음 음소거 토글 |
| **F11** | 전체 화면 토글 |
| **R** | (엔딩 화면) 다시하기 |
| 1~3 | (1회 클리어 후) 엔딩 화면에서 진/노멀/배드 감상 |

## 핵심 특징

- **읽고 판단하는 밭 관리** — 정답을 알려주지 않습니다. 수분·건강·잡초·해충·배수·스트레스
  여섯 스탯과 밭의 '증상'을 읽고 스스로 무엇이 필요한지 판단합니다.
- **기다림의 게임** — 작물은 제대로 된 돌봄과, 밭이 평온할 때의 '기다리기'에서만 자랍니다.
  '행동하기'와 '기다리기'는 분리되어 있고, 쉬지 않고 5턴 내리 일하면 강제로 돌발 상황을
  30초 안에 수습해야 합니다. 게임의 주제(인내)가 곧 공략법입니다.
- **태도로 갈리는 3개의 엔딩** — 점수가 아니라 인내·공감·이해가 결말(진·노멀·배드)을 결정하고,
  끝내 밭을 잃으면 '시듦'으로 마무리됩니다. 결과 화면이 '왜 이 엔딩에 이르렀는지'를 알려줍니다.
- **여러 작물** — 아무 엔딩이나 한 번 보면 사과나무·감자·벼가 열립니다. 작물마다 밭의 성질과
  버튼 이름, 미니게임(가지치기·굼벵이 잡기·물 대기·거름 주기)이 달라집니다.
- **악)몽중농원** — 진엔딩을 보면 열리는 지옥 모드. 검붉은 밭과 확장된 화면(960×720)에서,
  남긴 음식을 다시 길러 거두어야 꿈에서 놓여납니다.
- **저장 / 갤러리** — 수동 저장·이어하기·자동 저장(On/Off), 그리고 본 엔딩과 겪은 사건·회상을
  모아 보는 추억 갤러리.
- **절차적 사운드** — 모든 효과음·배경음을 numpy로 런타임 합성합니다(외부 음원 파일 0개).
- **마인크래프트풍 엔딩 크레딧** — 별무리 아래로 두 목소리의 시가 흐르는 스크롤 크레딧.

## 코드 구조

```
main.py             게임 루프 · 씬 전환 · 씬별 배경음 매핑

  game_state.py     전역 상태, 엔딩 판정, 텍스트 데이터(회상·깨달음·날씨 등)
  audio.py          numpy 절차적 오디오 엔진 (효과음 9종 · 배경음 3종)
  assets.py         픽셀 스프라이트 생성, 폰트, 배경 렌더링
  ui.py             패널·버튼·미터 등 공용 UI 위젯
scenes/             name_input · intro · farm · stage1~4 · memory ·
                    epiphany · story_choice · father_day · ending · transition
```
