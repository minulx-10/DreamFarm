# 몽중농원 🥕 (DreamFarm)

> *밀어낸 당근 한 조각에는, 누군가의 새벽이 들어 있었다.*

식탁 위 당근 반찬을 말없이 밀어낸 아이가, 그날 밤 꿈속의 당근밭에서 눈을 뜬다.
**"네가 밀어낸 것을, 이번에는 네 손으로 길러 보아라."**
한 뿌리의 당근을 길러내는 과정을 통해 아버지의 노동과 사랑을 이해해 가는,
Pygame으로 만든 감성 내러티브 농장 게임입니다.

---

## 실행 방법

```bash
pip install pygame numpy
python main.py
```

- Python 3.10+ / pygame(-ce) / numpy 필요
- 픽셀 폰트(Galmuri11)는 저장소에 포함되어 있으며, 없을 경우 자동 다운로드합니다.
- 오디오 장치가 없는 환경에서도 자동으로 무음 처리되어 정상 실행됩니다.

## 조작법

| 입력 | 동작 |
|------|------|
| 마우스 클릭 | 행동 선택, 드래그(정리·물주기·수확), 대화 진행 |
| 스페이스바 | 텍스트 진행 / 넘기기 |
| **M** | 배경음·효과음 음소거 토글 |
| **R** | (엔딩 화면) 다시하기 |
| 1~7 | (1회 클리어 후) 엔딩 갤러리 — 각 엔딩 감상 |

## 핵심 특징

- **읽고 판단하는 밭 관리** — 정답을 알려주지 않습니다. 수분·건강·잡초·해충·배수·스트레스
  여섯 스탯과 밭의 '증상'을 읽고 스스로 무엇이 필요한지 판단합니다.
- **기다림의 게임** — 작물은 제대로 된 돌봄과, 밭이 평온할 때의 '기다리기'에서만 자랍니다.
  조급함은 당근을 상하게 합니다. 게임의 주제(인내)가 곧 공략법입니다.
- **태도로 갈리는 7개의 엔딩** — 점수가 아니라 인내·공감·세심함·회복력이 결말을 결정합니다.
  결과 화면이 '왜 이 엔딩에 이르렀는지'를 알려줍니다.
- **절차적 사운드** — 모든 효과음·배경음을 numpy로 런타임 합성합니다(외부 음원 파일 0개).
- **연출 시스템** — 날씨·계절 변화, 깨달음(epiphany) 연출, 회상, 침묵의 선물,
  그리고 아버지의 시점으로 같은 밭을 돌보는 챕터.
- **돌발 미니게임** — 씨앗 고르기 / 물주기 / 해충 잡기 / 수확하기.
