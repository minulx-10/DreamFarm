# 기여 안내 (Contributing) — 몽중농원 (DreamFarm)

가장 환영하는 기여는 **번역**입니다. 게임을 여러분의 언어로 즐길 수 있게 도와주세요. 🌍

## 번역하기 (Translations)

번역은 별도 서비스 없이 **GitHub에서 직접** 합니다. 정본은 `core/i18n_data.py`, 번역 파일은
`core/locales/*.json` 입니다. 자세한 구조·규칙은 [`LOCALIZATION.md`](LOCALIZATION.md) 참고.

**기존 언어(예: 영어) 다듬기**
1. `core/locales/en.json`에서 값을 수정합니다 (`"한국어 원문": "번역"` 형태, **키는 그대로 두고 값만**).
2. `{자리표시자}`·`\n`·기호(`·` `—` `%`)는 **그대로 보존**하세요.
3. Pull Request를 올립니다.

**새 언어 추가 (예: 일본어 `ja`)**
```bash
python tools/i18n_export.py --add ja     # core/locales/ja.json 을 영어 값으로 프리필 생성
```
1. `core/locales/ja.json`의 값을 해당 언어로 번역합니다. (안 고친 키는 자동으로 영어로 폴백되니
   **부분 번역도 괜찮습니다.**)
2. `core/i18n.py`의 `_LANGS`에 `"ja"`를 추가하면 설정에서 전환됩니다.
3. Pull Request를 올립니다.

> 폰트 참고: 새 문자권(일본어 가나·한자 등)은 번들 폰트 **Galmuri11**이 해당 글리프를 포함하는지
> 확인이 필요할 수 있습니다.

## 자동 검사 (CI)

번역 관련 파일을 건드리는 PR은 GitHub Actions가 `python tools/i18n_export.py --check`로 **JSON 유효성과
원문에 없는 '유령 키'**를 검사합니다. 올리기 전에 로컬에서 같은 명령으로 확인할 수 있습니다.

## 코드 기여 (Bug fixes 등)

버그 수정·개선도 PR로 환영합니다. 큰 변경은 먼저 이슈로 상의해 주세요. 게임 구조는
[`HANDOFF.md`](HANDOFF.md)를 참고하세요.
